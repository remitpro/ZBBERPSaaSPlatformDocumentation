    Project Overview¶

    Implementation of a comprehensive Zero-Based Budgeting module integrated with the Bank's General Ledger system to automate and streamline the annual budgeting process, replacing the current Excel-based approach.

    Timeline: 12 months (5 phases)
    Budget: $3,638,000
    Spent to Date: $1,578,002 (43%, 0.4% under budget)
    Team Size: 16 members (13 core + 3 QA)

    Phase 2 Preview: Core Budget Module (Months 4-6)¶

    Objectives¶

    • Develop Loan Portfolio Module with IFRS 9 provisioning
    • Develop Funding Module (deposits and borrowings)
    • Implement Financial Statements Engine (Income Statement, Balance Sheet, Cash Flow)
    • Build KPI Dashboard with 13+ KPIs

    Key Features¶

    Loan Portfolio Module: - 5 product types (Retail, SME, Corporate, Mortgage, Consumer) - PFI classification - Interest income calculations - IFRS 9 provisioning engine - Loan repricing assumptions - Early repayment rates

    Funding Module: - Deposit budgeting by product and maturity - Borrowing budgeting (interbank, bonds, central bank) - Interest expense calculations - Liquidity ratio monitoring

    Financial Statements & KPIs: - Income Statement (15 line items) - Balance Sheet (22 line items) - Cash Flow Statement (indirect method) - 16 KPIs with targets and alerts (13 original + 3 additional from workshop)
