    Key Features Implemented¶

    Multi-Tenancy¶

    • Row-Level Security (RLS) in PostgreSQL
    • Tenant isolation via app.current_tenant session variable
    • Automatic filtering by tenant_id
    • Subscription plans (free, standard, professional, enterprise)

    Security¶

    • RBAC with 6 roles (admin, budget_manager, budget_user, approver, viewer, auditor)
    • MFA support (TOTP)
    • Password security (bcrypt, failed attempts, account locking)
    • JWT tokens with refresh mechanism
    • Rate limiting (100 req/15min per IP)

    Infrastructure¶

    • Multi-AZ deployment for high availability
    • Auto-scaling node groups (compute, memory, GPU)
    • Encrypted storage (KMS)
    • Read replicas for database scaling
    • CloudWatch monitoring and alarms
